# SQL COMMANDS
## DDL (Data Definition Language)
Defining the data
* CREATE
* ALTER
* DROP
* RENAME
* TRUNCATE
* COMMENT
## DQL (Data Query Language)
Interacting with and moving through the data
* SELECT
## DML (Data Modification Language)
Modifying the data
* INSERT
* UPDATE
* DELETE
* MERGE
* CALL
* EXPLAIN PLAIN
* LOCK TABLE
## DCL (Data Control Language)
Granting and revoking access to the data
* GRANT
* REVOKE
